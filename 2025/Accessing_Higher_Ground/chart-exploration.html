Chart Exploration

Primed with the automatic description, users have a mental model of the data. They can explore charts whole-to-part, filling in pieces, rather than part-to-whole.

This is the modern equivalent of providing a data table for detailed exploration of a dataset.

Full keyboard navigation

You can explore each series, and each datapoint.

Arrow key navigation:

  • `Up` & `Down`: move between series
  • `Left` & `Right`: move between datapoints
  • Different patterns for more complex charts types

Hot key navigation:

  • Jump to high and low points, for chart or series
  • Activate specific modes
  • Select datapoints
  • Do specific actions

Queries

  • `Q` key
  • Provides more details about a datapoint
  • Compares all selected datapoints

Annotations

  • Automatically added by AI for notable points
    • Intersections
    • Highs and lows
    • Start and end points of a series
  • Author can add manually, for explanatory charts
  • End user can add manually, to aid exploration and reasoning
  • On our roadmap: securely sharing annotations

Trend Navigation

  • You can explore the trend and each datapoint of each series
  • Other key features include
    • Other key features include intersections between lines
    • The dominant series
  • Intersections are described:
    • In the series description
    • During navigation

Cluster Navigation

  • Scatterplots may have dozens, hundreds, or thousands of datapoints
  • Unworkable to explore them sequentially
  • We provide cluster navigation
  • We provide cluster navigation
