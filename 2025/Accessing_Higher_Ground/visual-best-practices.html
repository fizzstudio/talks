Visual Chart Best Practices for Cognitive Load

Simple improvements to aid cognitive load and satisfy WCAG.

  • Using symbols for lines and datapoints
  • Direct labeling
  • Gaps between datapoints

Each of these draws on specific tested principles of Cognitive Load Theory, and on perception science.

Symbols

  • Aids with easy recognition
  • Satisfies WCAG success criteron 1.4.1: Use of Color
    • "Color is not the only way of distinguishing information."

Direct Labels

  • Much faster association
  • Helps people with reduced memory
  • Avoids Cognitive Load Theory's "split attention effect"
