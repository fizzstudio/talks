Colors & Patterns

Most common issue raised

  • Brand colors
  • Color contrast ratio
  • Not mathematically possible to have color contrast between more than 3 colors

Technical issues with color contrast algorithm

  • Not mathematically possible to have color contrast between more than 3 colors
  • Intended for text content, not graphical content
  • Developed in the 1990s for different monitor technology using a different color space, with a small informal study
  • Unclear who it's for:
    • People with color blindness? Which kind?
    • People with low vision?
      • Visual accuity is not measured with color contrast
    • People with cataracts?
  • Unintuitive false positives and negatives.
  • But… we need to comply with WCAG 2.1 for legal reasons. So let's do it smartly.

Patterns

  • Suggestion from W3C WCAG 2.1 Techniques
  • Technique G111: Using color and pattern
    • "patterns are included to convey the same information in a manner that does not depend on color.."

Color blind palettes

  • One size does not fit all… and can even be mutually exclusive!
  • Use smart and attractive defaults
  • Let the user customize colors for their own needs

Low Vision Mode

  • Switches to darkmode
  • Goes full screen
  • Increases font size and line thickness
  • Uses conventional white+black focus ring
  • Changes to primary-color palette
  • Makes popups static
  • All personalizable
  • Some low vision users won't even need magnifier
