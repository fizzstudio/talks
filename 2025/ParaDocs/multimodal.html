Multimodality

Simple improvements to aid cognitive load and satisfy WCAG.

  • Using symbols for lines and datapoints
  • Direct labeling
  • Gaps between datapoints

Sonification

  • Good for
    • Getting a gist
    • Identifying trends or outliers
  • Increases engagement
    • Grabs and keeps attention
    • Encourages students in data exploration
  • Delightful
    • Makes data fun
    • Provides novelty, breaks up long sections of text
    • Add aesthetic appeal

More complex data?

  • Not great for precision
    • As with color, we have difficulty mapping semantics to small differences

Sonification limitations

  • Ephemeral, and can be time-intensive
  • Often requires (too much?) training for real exploration
  • Use sparingly to manage attention, like animation
  • Need text for precision: visual, voice, or braille

SparkBraille

  • Uses a common refreshable braille display
  • Provides a tactile gist of the data in the braille cells
    • Two datapoints per braille cell
  • User can interact with the line with the cursor routing keys
