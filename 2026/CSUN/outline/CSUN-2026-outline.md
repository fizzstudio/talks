## Doug (intro)

- Introduce the topic
- Show how a story is more convincing and aids retention longer than raw numbers.
- "Charts are complex. There is not a single data story, there are many."
- "Disclaimer: we support all the basic chart types, but since stories are stronger with a time element, we're mostly showing line charts in this presentation."


## Charlotte (Basic accesibility and lead into Tour Guide)

- Charlotte is a native screen reader and magnifier user
- "Without an accessible platform, no accessible data stories are possible"
- Demonstrates how ParaCharts generates descriptions at the chart level, for each series, and for each notable datapoint.
- Shows trend mode, where we break a line chart down into its rising and falling sequences, each with its own description, reducing a line with 29 datapoints into 4 structured and described sequences.
- "By adding rich accessible navigation, we realized the potential for telling richer visual stories as well."


## Doug (Tour Guide)

- Demos basics of Tour Guide, how it syncs text and chart highlights, and self-voicing.
- "Obviously, you can see how this is better for people who have dyslexia, who often have symbol decoding issues. But it's also useful for anyone. It breaks down the description into chunks where each part is reinforced visually and textually. This reduces cognitive load."
- Show how this is self-paced with keyboard, and can be used with or without self-voicing.


## Ryan (Sharing stories)

- "This technique works well when you're on the web, where you can have full interactivity. But that's not possible for static media like PDF, social media posts, email, some LMSes, and print. So we made a portable solution."
- "When you generate a chart, we give you a static version, with a description that you can embed as alt text or a caption, and you can link back to a full interactive chart."
- Shows basic authoring tool, with different chart types, with "Export" options.


## Simon (Authoring stories)

- "We realize you may have a specific story to tell, which isn't covered by our automated narratives. So we built a prototype tool that can help you compose a narrative about the chart, and tie it to highlights on the chart."
- Demos TourGuider


## Tim (Scaffolding)

- "Tour Guide enables scaffolding information, establishing context and showing each piece of information in turn. Another way we do this is with scrollytelling"
- Demos scrollytelling.
- "We developed ParaChart as a framework. We think it could be used for scaffolding content, or scaffolding skills. We're especially interested to see how educators would use this, to save time."
